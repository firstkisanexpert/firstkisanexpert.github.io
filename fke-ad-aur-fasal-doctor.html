📢 Ad System

Google AdSense यहां दिखेगा
(नीचे बताए कोड में अपना AdSense Publisher ID / Ad Slot ID डालें)
Setup:
1) AdSense: Google AdSense अकाउंट बनाओ → अपना ca-pub-XXXX ID लो → नीचे <head> में AdSense स्क्रिप्ट डालो, और adsense-block वाली जगह पर असली <ins class="adsbygoogle"> टैग डाल दो (कोड में कमेंट में लिखा है)।
2) अपने बैनर: नीचे JS में bannerAds लिस्ट में जितने चाहें बैनर (image + link) जोड़ दो — अपने आप rotate होते रहेंगे।

🌾 Fasal Doctor - रोग व मिट्टी जांच चैट

नमस्ते! अपने पौधे/खेत की फोटो खींचो या अपलोड करो, और बताओ क्या दिक्कत लग रही है। मैं बता दूंगा कि कौन सा रोग है, कौन सा कीड़ा है, मिट्टी में क्या कमी हो सकती है, और क्या करना चाहिए।
ज़रूरी सेटअप (सिर्फ एक बार, बिल्कुल फ्री, कार्ड नहीं चाहिए):
1) aistudio.google.com पर जाओ, Google अकाउंट से sign in करो
2) "Get API key" पर क्लिक करो → "Create API key" दबाओ → key कॉपी कर लो
3) नीचे कोड में GEMINI_API_KEY वाली लाइन में यह key paste कर दो
4) फाइल GitHub Pages पर अपलोड कर दो — बस, हो गया

रोज़ाना करीब 1,500 सवाल/फोटो तक बिल्कुल मुफ्त चलेंगे — आपके चैनल के लिए बहुत ज़्यादा है।
